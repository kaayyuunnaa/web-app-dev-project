# BrightSmile Dental Center

A fully responsive multi-page website for BrightSmile Dental Center, built with HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

## Pages
- Home
- About Us
- Services
- Our Team
- Gallery
- Blog
- Contact

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5

## Features
- Responsive design (desktop, tablet, mobile)
- Contact form with JavaScript validation
- Dark/light mode toggle
- Filterable services gallery
